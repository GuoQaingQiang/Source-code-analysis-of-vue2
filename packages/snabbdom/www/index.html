diff function

新虚拟dom和老虚拟dom进行diff精细化比较，算出如何最小量更新，最后反映在真正的dom上

  • 最小量更新，key起到很大的作用
  • 只有同一个虚拟节点（选择器相同，key相同），才进行精细化比较
  • 只进行同层比较，不会跨层比较
  • 虚拟dom变成真实dom，涵盖在diff算法中
